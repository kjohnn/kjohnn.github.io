Instrument Voltage Transformer

The voltage transformer is used to measure high voltage on the primary terminals. Measuring the low voltage on the secondary terminals let you know the primary terminals voltage if the ratio is known.

zero burden error

ε0 = - Z1 / Z0 * Cos( φ1 - γ ) * 100 [%]

θ0 = - Z1 / Z0 * Sin( φ1 - γ ) * 3438 [分]

rated burden error

εb = - Z12 / Z * Cos( φ12 - φ ) * 100 [%]

θb = - Z12 / Z * Sin( φ12 - φ ) * 3438 [分]

誤差円線図

burden error

ε = ε0 - R / Z * { ( ε0 - εr ) * Cos φ + 0.0291 * ( θ0 - θr ) * Sin φ } [%]

θ = θ0 - R / Z * { ( θ0 - θr ) * Cos φ + 34.38 * ( ε0 - εr ) * Sin φ } [分]
